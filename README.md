# qk-release

> A beautiful javascript tool for quick release

## ✨ Feature

- One common and quick tool for release

## 🚀 Start

### install

```bash
npm i -D qk-release # OR yarn add -D qk-release
```

```js
  "scripts": {
    "release": "qkrelease <--skipBuild | --skipLog>"
  },
```

```bash
npm run release # OR yarn release
```
